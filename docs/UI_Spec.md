# AI 면접 프로그램 UI 및 기능 명세서 (UI Specification)

## 1. 공통 기능
* **로그인/회원가입:**
    * ID(Email)/PW 기반 자체 인증.
    * 역할 구분: 지원자(Candidate) vs 관리자(Admin).

## 2. 지원자(Candidate) UI
### 2.1 면접 설정 (Setup Page)
* **직무 선택:** Backend, Frontend, Data Science 등.
* **서류 제출:**
    * 이력서 업로드 (PDF) -> 텍스트 추출 및 파싱.
    * (Option) 기업 인재상/JD 업로드.
* **환경 테스트:** 카메라, 마이크, 스피커 작동 확인 토글.

### 2.2 면접 진행 (Interview Room)
* **레이아웃:** 2분할 화면 (좌: 화상 / 우: 채팅 로그).
* **화상 영역:**
    * `WebRTC` 스트림으로 본인 얼굴 실시간 송출.
    * AI 면접관 아바타(이미지/영상) 표시.
* **대화 영역:**
    * 실시간 STT 자막 표시.
    * AI 질문 텍스트 표시.
* **제어:** "면접 종료" 버튼 (조기 종료 시 패널티 경고 팝업).

### 2.3 결과 리포트 (Report Page)
* **종합 점수:** Radar Chart (직무역량, 논리력, 소통, 태도).
* **상세 피드백:**
    * 질문별 모범 답안 vs 내 답변 비교.
    * Vision 분석 결과 (예: "질문 3에서 시선이 불안정했습니다").
    * 개선 가이드 제공.

## 3. 관리자(Admin) UI (Phase 2 구현 예정)
* **채용 공고 관리:** 공고 등록, 수정, 삭제.
* **인재상 관리:** 직무별 평가 기준(프롬프트) 설정.
* **지원자 통계:** 전체 지원자의 점수 분포 및 합격률 대시보드.
